
# 💜 Naavya AI – Smart Glass Chatbot

A stylish AI-powered desktop chatbot built using Python with a modern lavender glass UI and Gemini API integration.

---

## ✨ Features

- 💬 Real-time AI responses using Gemini API  
- 🎨 Beautiful glass-style lavender UI  
- 💾 Chat saving system (persistent history)  
- 📤 Export chat to text file  
- 🌗 Dark/Light theme switching  
- ⚡ Fast and smooth user experience  
- 📦 Converted into standalone desktop app (.exe)

##  🚀 How to Run

1. Clone the repository:
Run the following commands in your terminal:
```bash
git clone https://github.com/srivastavanaavya75-lgtm/naavya-ai-chatbot-.git
cd naavya-ai-chatbot-
pip install -r requirements.txt
python chatbot.py
```
